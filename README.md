## Instructions
For change app name you have edit this files:

`main.cpp`
```#c++#
const string APP_NAME = "Test App";
// Change this value in line 15
```

for Excecutable

`CMakeLists.txt`
```
SET(appName HtmlExc)
# You can change the HtmlExc for your app name in line 2
```
